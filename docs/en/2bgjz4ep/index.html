Text Plugins

                        • []

                          There are two types of plugins. Ones that return plain-text and ones that return json. Each gets their own setup in the config. This lets the engine pass the data to templates as either text or serialized data

                        • [x]

                          A plugin directory is defined in the config file at `dirs.plugins``

                        • [x]

                          Plugin sections are also defined in the config in the `json_plugins`` and `text_plugins`` sections.
