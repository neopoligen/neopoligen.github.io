    Inline Key Value Spans

    Overview

    This feature lets you create inline spans that start with a key/value pair.

    The initial use case is for a single letter span tag to send css style classes for the span

    <<c: alfa bravo|Lorem ipsum>>

    Templates can use that to apply the classes to output something like

    <span class="alfa bravo>Lorem ipsum</span>
    • [x]

      Inline Key/Value Spans work without extra attributes

    <<c: green|Lorem Ipsum>>

    [x] Inline Key/Value Spans work with extra attributes:

    <<c: green|Lorem Ipsum|id: alfa>>
