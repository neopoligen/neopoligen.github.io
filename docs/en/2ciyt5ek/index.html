Overview

Used to see if a page has a filter in a case-insensitive way. This can be use for things like:

{% if page.has_filter("home-page") %}
This is the home page
{% else %}
This is not the home page
{% endif %}

Details

  • []

    Provides `page.has_filter("some_filter")`` that returns either `true`` or `false``

  • []

    Matching is done case-insensitive
