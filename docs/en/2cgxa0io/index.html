Overview

Every theme should be set up with a logging macro that calls to the site. Usage looks like this in the templates:

{%- import "includes/macros.jinja" as macros -%}

{{ macros.log({"msg": "some message"}) }}

The value is always a `{}`` hashmap/dictionary. The `msg`` key will be displayed in the error text in the output.

Other values can be send for other debugging as necessary. For example:

{{- macros.log({"msg": "some message", "more_stuff": ["a", "b"]}) -}}

The macro it itself is set up in the `includes/macros.jinja`` file with:

{%- macro log(payload = {}) -%}
{{- site.log_from_template(payload|tojson) -}}
{%- endmacro -%}
