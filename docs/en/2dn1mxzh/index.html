I just learned you can string multiple find and replace patterns together in a single `sed`` command. Used it to convert Jinja style template tokens that use `{`` characters into `[`` which are a lot easier for me to read

Looks like a cat stepped on the keyboard:

sed -i "" 's|{{-|[@|g;s|{{|[@|g;s|-}}|@]|g;s|}}|@]|g;s|{%-|[!|g;s|{%|[!|g;s|-%}|!]|g;s|%}|!]|g;s|{#-|[#|g;s|{#|[#|g;s|-#}|#]|g;s|#}|#]|g'
