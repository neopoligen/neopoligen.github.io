Overview

Each MiniJinja template in the theme's sections must have a category.txt file in it that lists which category the section belongs to. If that file is missing an error message is show in the app during the build process

Requirements

  • []

    If a section doesn't have a `category.txt`` file an error is displayed in the app

  • []

    If the `category.txt`` file doesn't have a valid category or is otherwise invalid an error stating that is displayed in the app
