    Overview

    This feature creates a function that outputs every section on a page except for the ones defined in the request. The function is:

    page.place_everything_except()

    It takes either single section name inside quotes. For example:

    page.place_everything_except("title")

    Or multiple parameters in quotes separated by commas

    page.place_everything_except("title", "p")
