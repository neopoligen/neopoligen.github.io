This is an early prototype of the site, lots of stuff is broken and somethings are out of date. That's the joy of buliding in public.

Syntax highlight is applied to code blocks when the first attribute is the name of a language. For example

-- code
-- rust

The output looks like this:

const helloWorld = () => {
    console.log("Hello, World")
}

document.addEventListener(
    "DOMContentLoaded", helloWorld
)

Each line is wrapped with a span tag that has the class `code_line``. That's how the line numbers are applied with this CSS:

.code_lines {
  counter-reset: line;
}

.code_line {
  counter-increment: line;
}

.code_line:before {
    content: counter(line);
    color: green;
    padding-right: 0.4rem;
}
