    Output Unknown Spans

    Overview

    This feature allows for unknown inline spans to exist. If one is found it gets converted into an element with the same name in the output HTML. For example:

    <<unknown_name|Lorem Ipsum>>

    Becomes:

    <unknown_name>Lorem Ipsum</unknown_name>
