Overview

This feature creates a default site title file in the config to be used as a final fallback if no other title can be determined for a page

Details

  • []

    Add a `site-title.txt`` config file to the `_configuration`` directory

  • []

    Use the value of the `site-title.txt`` file as the last fallback for page titles if no other one can be determined
