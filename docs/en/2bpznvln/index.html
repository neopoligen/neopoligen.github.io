Overview

This feature creates a `id()`` function on the `Page`` object that can be called on both the server side and in the templates

  • [x]

    Calling `.id()`` in the server code returns an `Option

  • [x]

    The id is pulled from the `-- id`` attribute of the `-- metadata`` section

  • [x]

    If `-- id`` exists it's returned as `Some(ID)``

  • [x]

    If an `-- id`` does not exist, the value returned is `None``
