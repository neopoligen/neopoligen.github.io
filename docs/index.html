How It Works

Neopoligen sites are made from text files that look like this:

-- title
-- subtitle: An app for making websites

Neopoligen

Neopoligen is a website builder.

It comes with lots of features built in. It's also 
very customizable. You can use it as-is or dig in 
and customize it completely.

Those files get combined with templates to build a static site that can be viewed locally and deployed anywhere that offers static site hosting.

Sections And Spans

Files are made up of "Sections" and "Spans". Sections are things like `-- title`` in the above example. Spans looks like this ` some text ``
